# 05 - Billing and Costs

gh0stcloud cost measurement is based on runtime usage and customer allocation,
not on comments in GitOps manifests.

Use gh0stportal Billing to review:

- trial status and deadlines
- issued invoices
- payment authorization state
- manual or automatic collection settings
- cost timeline and allocation views

For this example, the cost-relevant resources are:

- one small deployment
- one service
- one ingress route
- optional PVC in the advanced/professional overlays
- optional additional replica in the professional overlay
